# 消融实验设计说明

## 实验设计

本消融研究采用 **"独立对照 + 最优路径叠加"** 的混合设计：

**第一阶段（独立筛选）**：在 v0 Baseline 基础上，对提示词(v1)、生成参数(v2)、路由词(v3)、生成约束(v4) 分别进行单变量改动，快速量化每个优化方向的独立边际贡献。每个版本 = v0 + 单一变量改动，其他参数完全一致。

**第二阶段（最优组合）**：基于第一阶段结果，选取正向效果最显著的改动（v2、v3、v4），按逻辑顺序叠加形成 vFinal，验证组合后的协同效应与潜在副作用。

此设计兼顾了**归因清晰度**（每个变量的独立贡献可量化）与**叙事连贯性**（最终版本是有据可依的最优组合）。

---

## 版本定义

| 版本 | 改动变量 | 实验意图 | 具体改动 |
|------|----------|----------|----------|
| **v0** | 无（Baseline） | 建立基线 | 原始迁移版本，最简医疗Prompt |
| **v1** | 仅提示词措辞 | 验证纯提示词工程的作用 | 优化领域措辞，增加循证医学要求 |
| **v2** | 仅生成参数 | 验证温度/长度对质量的影响 | temperature 0.3→0.1，max_tokens 500→800 |
| **v3** | 仅路由关键词 | 验证意图识别的改进空间 | 启发式关键词从3个扩展到17个 |
| **v4** | 仅生成约束 | 验证结构化约束的强制力 | 强制"问题分析/专业解答/注意事项/就医建议"四段式 + 免责声明 |
| **vFinal** | v2+v3+v4 组合 | 验证最优组合的协同效应 | 叠加所有正向改动（排除v1） |

---

## 关键实验发现

### 发现1：v1 反直觉结果 — 纯措辞优化不够

v1 仅优化了提示词的领域措辞（增加"循证医学""请在医生指导下用药"等表述），但医疗安全率反而从 20.0% 下降到 13.3%。

**归因分析**：措辞优化让模型更"自信"地回答，反而减少了免责声明的输出频率。这证明在医疗场景下，**仅靠措辞暗示不够，需要显式的结构化约束（v4）才能可靠地提升安全性**。

**面试话术**："v1 是一个反直觉的发现。我原以为优化提示词措辞就能提升安全率，但数据告诉我恰恰相反。这让我意识到医疗场景需要的是硬约束而非软暗示，于是设计了 v4 的结构化强制方案。"

### 发现2：v4 副作用与协同效应

v4 单独使用时，医疗安全率大幅提升（+66.7pp），但"无诊断性断言率"从 100% 下降到 80%。然而在 vFinal 中，该副作用完全消失（恢复到 100%）。

**归因分析**：严格的结构化约束迫使模型在"就医建议"段落中给出更明确的判断，偶尔越界为诊断性表述。但当与 v2（低温度）组合后，模型的随机性降低，不再产生越界表述。

**面试话术**："v4 单独跑出了一个副作用——严格约束导致诊断性断言率上升。但在 vFinal 中这个副作用消失了，因为低温度抑制了模型的'过度自信'。这说明系统调优不是单点优化，而是需要寻找参数组合的最优解。"

### 发现3：零回归的工程严谨性

Negative Churn 分析显示：vFinal 相比 v0，修复了 8 个 Badcase，引入 0 个新 Badcase。回归通过率 100%。

**面试话术**："除了看平均指标提升，我特别关注版本间的回归分析。数据显示零回归、8个修复，证明迭代过程没有引入新问题。这是我理解的工程严谨性——不仅要让系统变好，还要确保不会变差。"

## 评估体系设计原则

### 数据采集与指标计算分离

```
数据采集层（昂贵，需调用LLM API）
  → ablation_results.json（原始问答对 + 路由决策 + 元数据）

指标计算层（廉价，纯本地计算）
  → scripts/compute_metrics.py --input ablation_results.json
  → 新增指标只需修改此脚本，零API成本
```

### 重评策略

| 变更场景 | 是否需要重调API | 应对方式 |
|----------|:---:|----------|
| 新增评估指标 | 否 | 重跑 compute_metrics.py |
| 扩大样本量 | 是 | 断点续传 + 增量采集 |
| Agent代码逻辑变更 | 是 | 必须全量重跑，不可混用 |

### 信源质量监控

采用 `high_authority` / `low_authority` 关键词规则作为**趋势监控的代理指标**。该指标不用于精确判断单条回答的信源质量，而是用于监控版本间的信源偏移趋势。
